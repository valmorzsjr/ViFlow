VidFlow
O VidFlow é uma plataforma de streaming de vídeos, desenvolvida para praticar o consumo de APIs, manipulação assíncrona do DOM e filtros dinâmicos com JavaScript.

Tecnologias Utilizadas
HTML5 e CSS3 para a estrutura e estilização responsiva.

JavaScript (ES6+) para toda a lógica da aplicação.

Fetch API com async/await para consumo de dados.

My JSON Server para simular um banco de dados (API).

Vercel para hospedagem e deploy.

Funcionalidades
Renderização Dinâmica: Os vídeos são carregados de um arquivo JSON externo.

Barra de Pesquisa: Filtra os vídeos por título em tempo real conforme você digita.

Filtro por Categorias: Botões que permitem filtrar o conteúdo (Front-end, Mobile, Data Science, etc.).

Tratamento de Erros: Exibe mensagens amigáveis caso a API falhe ao carregar.
